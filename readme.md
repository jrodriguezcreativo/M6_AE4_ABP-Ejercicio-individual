# M6_AE4_ABP-Ejercicio individual

## Portafolio Personal — Express + Handlebars

### Descripción
Pequeño portafolio personal servido con Express y plantillas Handlebars (hbs).  
Incluye página principal con nombre y lista de 3 proyectos, página About, página Projects con detalles, parciales header/footer y helper que muestra el año actual.

### Objetivos
- Configurar Express y Handlebars.  
- Renderizar vistas dinámicas desde el servidor.  
- Usar parciales y helper custom en Handlebars.  
- Servir archivos estáticos desde `public/`.

### Estructura del proyecto
```
M6_AE4_ABP-Ejercicio individual
|-- readme.md
|-- .gitignore
|-- package.json
|-- app.js
|-- public/
  |-- css/
    |-- style.css
  |-- img/
|-- views/
  |-- partials/
    |-- header.hbs
    |-- footer.hbs
  |-- home.hbs
  |-- about.hbs
  |-- projects.hbs
  |-- 404.hbs
```


### Instrucciones rápidas
1. Ejecutar `npm install`.  
2. Levantar servidor con `node app.js`.  
3. Abrir `http://localhost:3000` en el navegador.
